C++ Projects


Drone Coordinates

I implemented algorithms about which order a drone should deliver packages in. The first algorithm used the MST, but with a twist because the drone could only deliver to a certain section of the coordinates from the x-axis or y-axis. The second algorithm focused on efficiently running the algorithm instead of finding the optimal path. I used the arbitrary insertion algorithm to find a more optimal path. I implemented the last algorithm using recursion, checking every path but stopping early if the path was not promising.

Log entries

I kept track of various log entries by using multiple maps. I am able to output the number of entries that contain a certain key or category word. Other features the user can search in the log entries are which ones fall at a specific time or between two times. The user can also add or remove entries.

Sith and Jedi Battles

I used priority queues to make sure the Sith and Jedi only battled when the power level of the Sith army was greater than the Jedi army, and to ensure the highest power level Sith army always battled first. I also kept track of the battles with the most power level difference, how many soldiers were lost for each general, and the running median with another priority queue.

Maze Exploration

I used a stack and a queue to search through a maze for the Countess. I was able to explore every path until the Countess was found or there was no more path to search. My algorithm kept track of the path and outputted the directions to find the Countess.

Classifying Piazza Posts

I used machine learning by training my program with past piazza posts in order to classify new piazza posts. This algorithm looked at how past posts were classified, and looked at the words that were used in the questions posted. Then, my program uses probability to choose how the unclassified posts should be classified.

Euchre

For a class project, I implemented the card game Euchre. There is a simple machine player option that optimizes their move each round based on their cards and the first card placed.
